DARK PIVAULT

Die Vision

Dark PiVault wurde entwickelt, um die Lücke zwischen mobiler Datenverfügbarkeit und maximaler Sicherheit zu schließen. Herkömmliche Cloud-Speicher erfordern Vertrauen in Drittanbieter. PiVault erfordert nur Vertrauen in deinen eigenen Code und deine eigene Hardware.

Der RAM-Only Ansatz

Ein entscheidender Sicherheitsaspekt von PiVault ist die flüchtige Speicherung. Während des Verschlüsselungsvorgangs werden keine Daten auf der SD-Karte des Raspberry Pi zwischengelagert. Alles geschieht in Echtzeit innerhalb des 8GB LPDDR4X Arbeitsspeichers. Sobald der Strom unterbrochen wird, existieren keine Spuren der unverarbeiteten Daten mehr.

Core. Security Specs

  • > Algorithm: AES-256-GCM (Authenticated Encryption)
  • > Key Derivation: PBKDF2 with SHA-256
  • > Protocol: Stateless Flask-REST API
  • > Transfer: Chunked Data-Streaming via Requests-Toolbelt
